# Week 4: Backend Setup with Node.js & Express.js

## Objective
Establish the server-side environment for the Portfolio project. Setup a Node.js runtime, construct an Express.js server, and construct API endpoints to receive data.

## Learning Outcomes
*   Configured the Node.js project environment using NPM package management.
*   Learned the core concepts of Node.js HTTP servers.
*   Leveraged the Express framework to create robust APIs with modular routing rules.
*   Implemented essential express middlewares (`cors`, `express.json`) to handle Cross-Origin Resource Sharing and JSON payloads.
*   Created HTTP GET and POST request route handlers.

## Folder Deliverables
- `package.json`: Manages the project metadata and external libraries (`express`, `cors`).
- `server.js`: Express server implementation configuring endpoints.

## How to Test
1. Open your terminal in this directory.
2. Run `npm install` to download dependencies.
3. Run `npm start` to run the server.
4. Visit `http://localhost:5000/api/status` in your browser to verify it's active.
