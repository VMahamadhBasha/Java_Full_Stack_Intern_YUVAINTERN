# Week 2: Responsive CSS Design & Visual Identity

## Objective
Apply CSS3 styles to create an Obsidian Dark aesthetic, utilizing modern features such as design tokens (CSS variables), grid/flexbox layouts, custom typography, backdrop-filters (glassmorphism), and glow canvas orbs.

## Learning Outcomes
*   Established a cohesive design language using CSS variables.
*   Mastered responsive structural designs using standard CSS Flexbox and CSS Grid.
*   Learned to apply glassmorphic principles (semi-translucent borders, backdrop blurring).
*   Configured typography styling using custom Google Fonts (Inter and Outfit).

## Folder Deliverables
- `index.html`: Layout adjusted to link to external styles.
- `index.css`: Stylesheet implementing the entire design system and responsiveness.

## How to View
Double-click `index.html` to preview the stylized, responsive frontend layout in your web browser.
