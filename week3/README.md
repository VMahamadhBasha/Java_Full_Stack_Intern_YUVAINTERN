# Week 3: Client-Side Interactivity with JavaScript (ES6)

## Objective
Enable a dynamic, interactive user experience on the frontend by using native DOM APIs to handle scrolling states, build responsive elements, animate headers, and filter content profiles.

## Learning Outcomes
*   Understood DOM event listeners (`DOMContentLoaded`, `scroll`, `click`).
*   Created a dynamic typing loop carousel for text animation without third-party plugins.
*   Programmed a client-side filter engine to highlight portfolio specializations (Java Full-Stack or ServiceNow).
*   Built a mobile drawer interface that dynamically toggles slide positioning classes.

## Folder Deliverables
- `index.html`: Base portfolio structure.
- `index.css`: Styles for animations, transitions, and mobile hamburger states.
- `index.js`: Script containing execution loops, listeners, and specialization filter operations.

## How to Run
Double-click `index.html` and test:
1. Scrolling to see the navigation header shrink.
2. Clicking the specialization filters to observe opacity changes.
3. Reducing screen width to check mobile menu activation.
