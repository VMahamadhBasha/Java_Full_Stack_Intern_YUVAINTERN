# Week 1: Project Initiation & Skeleton HTML Structure

## Objective
Establish the project foundation by planning the web application layout and building a structured, semantically correct HTML skeleton for the Portfolio website.

## Learning Outcomes
*   Understood semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
*   Designed a layout targeting a dual professional identity: Software Engineer (Java Full-Stack) and ServiceNow Developer.
*   Learned the importance of document structure, SEO meta tags, and accessibility elements (`lang`, `alt`, `label` association).

## Folder Deliverables
- `index.html`: Skeleton HTML file containing structured sections for the homepage, portfolio info, skills, projects, timeline, and contact form.

## How to View
Double-click `index.html` to open it in any standard web browser. At this stage, the content is unstyled but structured properly.
